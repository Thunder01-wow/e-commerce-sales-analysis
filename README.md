# E-Commerce Sales Analysis

## Project Overview
This project analyzes an e-commerce sales dataset to understand sales trends,
profitability, and customer behavior across different categories and segments.

## Objectives
- Analyze overall sales and profit performance
- Identify top-performing categories and sub-categories
- Study monthly sales trends
- Compare sales and profit across customer segments

## Tools Used
- Python
- Pandas
- NumPy
- Plotly

## Key Insights
- Technology category generates the highest sales
- Furniture shows lower profit margins
- Sales show seasonal trends
- Consumer segment dominates revenue

## Conclusion
This project demonstrates end-to-end exploratory data analysis using Python and Pandas.
